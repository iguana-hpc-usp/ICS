# ICS - IGUANA CLUSTER SYSTEM

You can install the Iguana in two ways:

1. Run the command below on your Ubuntu 20.04 (tested) to execute the installation script; or

```
apt install curl -y && curl -s https://raw.githubusercontent.com/iguana-hpc-usp/ICS/master/install.sh | bash -s --
```

2. Download the virtual machine ready, just import it into VirtualBox.

* [Host1](https://drive.google.com/file/d/1RvpZsCzN3P7N5o47-Ikoa1Ml_j34oStO/view?usp=sharing) - Click to download
